## Step 3: Visual Interaction Flows

### Entry Flow
1. Vehicle arrives
2. Slot allocated
3. Ticket generated
4. Slot marked as occupied

### Exit Flow
1. Ticket scanned
2. Fee calculated
3. Payment processed (with retries)
4. Receipt generated
5. Slot released
6. Ticket deactivated

### Admin Flow
- Add floor
- Add slot
- Update pricing
